Este laboratorio se centra en la exploración de distribuciones de probabilidad y la simulación estadística mediante Python. El objetivo es comprender los fundamentos matemáticos que sustentan las metodologías de ciencia de datos y aprendizaje automático.

Mediante técnicas de simulación y visualización, se analizó el comportamiento de variables aleatorias y distribuciones estadísticas.

## Objetivos
- Comprender las distribuciones de probabilidad fundamentales.
- Simular procesos aleatorios con Python.
- Visualizar el comportamiento estadístico de los datos simulados.
- Analizar resultados teóricos frente a empíricos.
- Fortalecer la intuición matemática para la ciencia de datos.

## Conceptos aplicados
- Distribuciones de probabilidad.
- Simulación de variables aleatorias.
- Visualización estadística.
- Comparación empírica vs. teórica.
- Conceptos de modelado matemático.

## El laboratorio demuestra
- Generar muestras aleatorias a partir de distribuciones estadísticas.
- Visualizar la forma y el comportamiento de las distribuciones.
- Comparar los resultados simulados con las predicciones teóricas.
- Analizar la variabilidad y la aleatoriedad en conjuntos de datos.
- Interpretar patrones estadísticos mediante representaciones gráficas.

## Conclusiones
- Las distribuciones simuladas convergen a las expectativas teóricas con un tamaño de muestra suficiente.
- La variabilidad disminuye a medida que aumenta el tamaño de la muestra.
- La visualización mejora la comprensión del comportamiento probabilístico.
- La simulación es una herramienta poderosa para validar supuestos estadísticos.
